# Slay the Spire: Show Boss Chest Tooltips

When viewing a boss chest, display all three relic tooltips.

Primarily for use with [Slay the Streamer](https://steamcommunity.com/sharedfiles/filedetails/?id=1610759491), so chat can see what modded relics do.
